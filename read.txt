
* fast api 설치 확인
pip show fastapi 

* uvicorn 설치 확인
pip show uvicorn

* 반드시 설치
pip install fastapi
pip install "uvicorn[standard]"
pip install pytz
pip install jinja2
pip install pymongo


* 한 번에 설치하는 코드
pip install fastapi uvicorn jinja2 pymongo pytz


python main.py로 실행!!
